# Задание 1: Скриншот с багами

```text
Найдите баги на приложенном скриншоте, приоритезируйте их и перечислите списком.
Приоритеты:
P0 - критический,
P1 - высокий,
P2 - средний,
P3 - низкий.
Пример: P0 - не открывается объявление из выдачи.
```

[![Скриншот с багами](bags.png)](bags.png)
***

**P0 (Блокирующий / Критический):**

```text
Критические дефекты базового функционала.
Пользователь не может найти товар, поиск выдает абсолютно нерелевантную выдачу,
ломается ключевой пользовательский путь.
Сломана базовая логика поиска.
Из-за этих багов фильтры бесполезны - люди перестают верить поиску.
```

P0 - Фильтр: Цена. Нарушение границ фильтрации. При установленном диапазоне 1 000–30 000 ₽ в выдаче отображаются объявления с ценой 500 и 40 000.

P0 - Фильтр: Локация. Выдача объявлений из других регионов. При поиске по Москве в результатах присутствует объявление из Новосибирска.

P0 - Фильтр: Пол. Игнорирование выбранного параметра. В фильтре слева активно значение "Мальчик", однако в выдаче присутствует объявление "Персидский котёнок девочка".

**P1 (Высокий):**

```text
Серьезные ошибки логики, навигации или выдачи.
Фичи работают некорректно, путают пользователя,
но приложение не падает, и есть обходные пути.
Нарушение логики категорий и ложные ошибки.
Здесь ломается структура каталога и UX страницы.
```

P1 - Каталог: Порода. Нерелевантная порода в выдаче. При выбранном фильтре "Персидская" в выдаче отображается объявление "Экзотический короткошёрстный котёнок", а также котенок бенгальской породы (третье фото в первом ряду не соответствует заголовку, но тут уже сложно определить пользователь перепутал или может есть проверка фото на породу...).

P1 - Модерация/Категория. Неверный тип товара. В выдаче живых животных отображается мягкая игрушка ("Плюшевый котёнок").

P1 - Навигация: Хлебные крошки. Нарушена иерархия. Неверный порядок категорий в шапке: "Главная > Животные > Персидская > Кошки". Ожидается: Главная > Животные > Кошки > Персидская".

P1 - UI: Состояние системы. Ложное сообщение об ошибке. На странице присутствует текст "Ошибка, обновите страницу", хотя контент успешно загружен и доступен для взаимодействия.

P1 - Фильтр: Порода. При поиске породы текстом не хватает явного чекбокса (сейчас пользователю может быть неочевидно, как применить найденную породу, а если он хочет выбрать несколько пород).

**P2 (Средний):**

```text
Ошибки UI/UX, некорректные тексты, сломанные второстепенные элементы.
Раздражает, но не блокирует покупку/продажу.
Влияют на удобство использования, но не блокируют основную задачу.
```

P2 - UI: Всплывающие подсказки. Неверный тултип на кнопке "Избранное". При наведении на иконку "сердечко" появляется подсказка "Не показывайте мне похожие объявления" вместо ожидаемой "Добавить в избранное и в сравнение".

P2 - Сортировка. Несоответствие гео-контекста. Переключатель радиуса поиска называется "Сначала из Можайска", несмотря на то, что геопозиция поиска - Москва. Ожидается "Сначала из Москвы".

P2 - Фильтр: Возраст. Некорректный нейминг для кошек. В фильтре возраста присутствует вариант "Щенок" вместо ожидаемого "Котенок".

P2 - Счетчики. Рассинхронизация количества объявлений. В заголовке указано "19" объявлений, а на кнопке пагинации/подгрузки снизу - "Показать 8 объявлений" (и визуально их тоже 8).

**P3 (Низкий):**

```text
Мелкие визуальные недочеты (съехавшие пиксели, отступы),
которые заметит только тестировщик или дизайнер.
Задачи в бэклог для исправления технического долга.
```

P3 - Вёрстка: Сетка. Разные отступы между карточками. Нарушена модульная сетка, расстояния между объявлениями в рядах неодинаковые.

P3 - Вёрстка: Текст ошибки. Отсутствует стилизация. Сообщение "Ошибка, обновите страницу" выведено мелким системным шрифтом поверх фона, без иконки или плашки (сломан UI-компонент).

P3 - Вёрстка: Фильтры. Блок фильтров не выровнен. Кнопки выбора ("Любой", "Мальчик", "Девочка" и другие блоки внизу) не выровнены по единой направляющей слева. (Заметила,что на Avito.ru тоже самое в фильтрах).

P3 - Вёрстка: Рекомендации. Обрезка контента. Блок "Подобрали для вас" не имеет отступа справа и визуально выезжает за пределы безопасной зоны экрана. Нет адаптации контента.

P3 - Фильтр: Окрас. Для домашних животных окрас является важным критерием выбора, стоит добавить этот параметр в фильтры.

P3 - Карточка объявлений в сетке: Дата публикации. В некоторых карточках присутствует дата публикации. а в некоторых нет. Нужно уточнить у аналитиков: это баг отображения или фича для продвигаемых/платных объявлений? (На Avito.ru нет дат в сетке выдачи).

P3 - Блок "Подобрали для вас". В блоке "Подобрали для вас" показываются котята других пород. Обычно рекомендательные системы ML специально подмешивают похожие, но другие категории для кросс-сейла, поэтому заводить это как баг нужно только после проверки логики работы рекомендаций.
